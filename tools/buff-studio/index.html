使用说明 / How to use
  1. ① 加图层：emoji / 本地图片 / 拖入 GIF·视频，每个素材一层，可叠多层做拼贴
  2. ③ 摆位置：合成台上拖动移动；顶部把手旋转、右下把手缩放；右侧还能调透明度 / 混合模式
  3. ④ 画板：随手画一个图标，「保存到素材」后在左侧「本地图片」中作为图层使用
  4. ⑤ 加动画：50 种循环无缝动画（脉冲 / 抖动 / 粒子迸发 / 悠扬乐符 / 落叶 / 萤火虫…），参数可调
  5. ⑦ 点生成：烘焙所有帧 → ffmpeg.wasm 编码带 alpha 的 WebM（首次下载 ~30MB，之后缓存）

生成的 .webm 可放进 OBR Suite 状态追踪的 public/buff-fx/，或作为 OBR Image item 的 url。
